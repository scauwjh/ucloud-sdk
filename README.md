ucloud-sdk
----------
快速接入ucloud api

maven 依赖
------------
* guava 19.0
* jodd-http 3.7.1
* alibaba fastjson 1.2.15
* commons-codec 1.10
* junit 3.8.1

usage:
------
```
<dependency>
    <groupId>me.keiwu</groupId>
    <artifactId>ucloud-sdk</artifactId>
    <version>1.0.0</version>
</dependency>
```
